============
Installation
============

..  _installation-composer:

Install with Composer
=====================

..  note:: 
    This is the recommended way to install this extension.

Install the extension via Composer:

..  code-block:: bash

    composer req oliverkroener/ok-exchange365-mailer

See also `Installing extensions, TYPO3 Getting started <https://docs.typo3.org/permalink/t3start:installing-extensions>`_.

..  _installation-classic:

Install in Classic Mode
=======================

.. warning::
    This is not the recommended way to install this extension, and **might involve manual steps** to add **dependencies**.

Or download the extension from `https://extensions.typo3.org/extension/ok_exchange365_mailer <https://extensions.typo3.org/extension/ok_exchange365_mailer>`_ and install it in
the Extension Manager.
